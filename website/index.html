Recording Details

Records lossless WAV files. Choose between Mix (system + mic), System Audio, or a specific input device in-app. Files are saved to your Music folder by default (path editable in the app). Filenames use the pattern AljoRecording_YYYYMMDD_HHMMSS.wav. System and mic gain are applied safely; recording is written with thread-safe buffering. Check the app log or UI for any write errors.

FAQ

Recording format: The app records lossless WAV files. Filenames use AljoRecording_YYYYMMDD_HHMMSS.wav.

Save location: By default recordings go to your Music folder; you can change the save path in-app and it's persisted between runs.

Source options: Choose Mix (system + mic), System Audio, or a specific input device from the recording panel inside the app.

Troubleshooting: If a recording fails to save, check the app log at %AppData%\AljoAudioBridge\app.log and ensure the target folder is writable.
